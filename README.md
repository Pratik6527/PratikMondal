# 🌹 Valentine's Proposal Website

A romantic, interactive website template designed for a special proposal.  
**Live Demo:** [Open `index.html` in your browser]

## ✨ Features
- **Hero Section**: Customizable photo and heartfelt message.
- **Interactive "YES" Button**: Triggers confetti, music, and a celebration modal.
- **Impossible "NO" Button**: Playfully runs away from the cursor/touch.
- **WhatsApp Integration**: Sends a pre-filled "I said YES!" message to your number.
- **Memory Gallery**: Upload photos directly in the browser to preview memories.
- **Mobile Responsive**: Looks great on all devices.

## 🚀 How to Use
1.  **Clone or Download** this repository.
2.  Open `index.html` in any web browser to view the site.
3.  **Customize**:
    -   **Photo**: Replace `assets/hero_image.png` with your own photo.
    -   **Music**: Replace `assets/music.mp3` with your song.
    -   **WhatsApp**: Edit `script.js` (line 104) to set your phone number.
4.  **Upload to GitHub (Manual)**:
    -   Go to your repository on GitHub.
    -   Click **Add file** > **Upload files**.
    -   Drag and drop all files from this folder.
    -   Click **Commit changes**.
